DATA CLEANING LOGIC
===================

    Input Excel File
    ┌─────────────────────────────────────────┐
    │ Row 1: Normal data                      │ ✓ KEEP
    │ Row 2: Contains "FOC" in column BV      │ ✗ REMOVE
    │ Row 3: Contains "test" in column H      │ ✗ REMOVE
    │ Row 4: Normal data                      │ ✓ KEEP
    │ Row 5: Contains "M88" in column BO      │ ✗ REMOVE
    │ Row 6: Normal data                      │ ✓ KEEP
    │ Row 7: Contains "GB Test" in column H   │ ✗ REMOVE
    │ Row 8: Normal data                      │ ✓ KEEP
    └─────────────────────────────────────────┘
                    │
                    ▼
    Output Excel File (*_CLEANED.xlsx)
    ┌─────────────────────────────────────────┐
    │ Row 1: Normal data                      │
    │ Row 2: Normal data                      │
    │ Row 3: Normal data                      │
    │ Row 4: Normal data                      │
    └─────────────────────────────────────────┘


COLUMN CHECKING PROCESS
========================

    For each row in Excel file:
    
    ┌─────────────────────┐
    │ Check Column BV     │
    │ (ShipmentID)        │
    │ Contains "FOC"?     │
    └──────────┬──────────┘
               │ YES → Mark row for removal
               │ NO ↓
    ┌─────────────────────┐
    │ Check Column H      │
    │ (Order)             │
    │ Contains test/M88/  │
    │ GB patterns?        │
    └──────────┬──────────┘
               │ YES → Mark row for removal
               │ NO ↓
    ┌─────────────────────┐
    │ Check Column I      │
    │ (Buyer PO Number)   │
    │ Contains test/FOC?  │
    └──────────┬──────────┘
               │ YES → Mark row for removal
               │ NO ↓
    ┌─────────────────────┐
    │ Check Column BO     │
    │ (Comment)           │
    │ Contains FOC/M88?   │
    └──────────┬──────────┘
               │ YES → Mark row for removal
               │ NO ↓
    ┌─────────────────────┐
    │ Keep this row       │
    └───────────────���─────┘

PATTERN MATCHING EXAMPLES
==========================

    Column H (Order) - Remove if contains:
    
    "test"
    ├── "test" ✗
    ├── "Test" ✗ (case insensitive)
    ├── "testing" ✗ (substring match)
    ├── "test123" ✗ (substring match)
    ├── "my test order" ✗ (substring match)
    └── "order" ✓ (no match, keep)
    
    "M88"
    ├── "M88" ✗
    ├─�� "m88" ✗ (case insensitive)
    ├── "M880123" ✗ (substring match)
    ├── "M88-ABC" ✗ (substring match)
    └── "M87" ✓ (no match, keep)
    
    "GB"
    ├── "GB" ✗
    ├── "gb" ✗ (case insensitive)
    ├── "GB Test" ✗
    ├── "GB Testing" ✗
    ├── "GB Order" ✗ (substring match)
    └── "Order" ✓ (no match, keep)

               ▼
